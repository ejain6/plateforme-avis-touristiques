# PACT – Plateforme d’Avis et Conseils Touristiques

## Présentation

PACT est une plateforme web permettant aux professionnels du tourisme de partager des établissements et activités, et aux visiteurs de rédiger des avis et de les consulter pour préparer leur séjour.
Projet réalisé en 2ème année de BUT Informatique, pour un client fictif joué par des professeurs.

## Technologies

#### Développement :
- HTML / CSS
- JavaScript
- PHP
- PostgreSQL

#### Outils :
- Jira
- Figma

## Fonctionnalités principales

- Consultation et filtrage des offres
- Affichage des offres sur une carte
- Consultation et publication d'avis touristiques
- Gestion des comptes utilisateurs
- Authentification à 2 facteurs
- Interface d'administration pour les professionnels
